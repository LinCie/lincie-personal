This site is the work. Not a portfolio in the conventional sense — no case studies formatted for a hiring committee, no metrics dressed as proof. LinCie is a place where engineering and design are treated as the same discipline expressed through different materials.

The architecture is deliberately minimal: Astro for static generation, Tailwind for a CSS token layer that enforces visual consistency without fighting the cascade, and GSAP reserved for motion that earns its complexity. Every dependency was chosen because it removes friction rather than adds capability for its own sake.

What you read here and what you see here are the same argument. The typography is set to read at length. The motion is quiet enough to notice only when it's absent. The structure holds without announcing itself.
